# Você Digital Propaganda Landing Page

Landing page de captura desenvolvida com Next.js (App Router), TypeScript estrito, Tailwind CSS, Radix UI e Framer Motion.

## Stack

- Next.js 16+
- React 19
- TypeScript
- Tailwind CSS 4
- Radix UI Slot
- Framer Motion

## Como rodar

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```

## Deploy

O projeto está pronto para deploy em Vercel ou Netlify. Para metadata absoluta em produção, defina:

```bash
NEXT_PUBLIC_SITE_URL=https://seu-dominio.com
```
